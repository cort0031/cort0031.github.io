Notes

Headings
The <h1></h1> tag is used for the most important heading of the page. Other levels of heading we can use <h2></h2>, <h3></h3>, <h4></h5>, and <h6></h6>.
Paragraphs
To add a paragraph we can use the <p></p> tag.
Quote
Using the <q></q> we can quote our content. I used the code from the Class' Github page
Emphasis
Using <em></em> we can put some emphasis to the text, so can change their voice or meaning for this text.
Important text
Using <strong></strong> we can tell this is important stuff, so the readers can put special attention or change their voice for this text.
